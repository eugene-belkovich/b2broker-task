# B2broker task

Websocket server is running on port 443 use `ws://localhost:443` to connect to it.

### Installation

```bash
$ npm run init
```

## Running the app

```bash
# local
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# watch mode
$ npm run test:watch

# test coverage
$ npm run test:cov
```
